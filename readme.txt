== Description ==

Compress your images with Exhale and make your site smaller by up to 80%. All of that without any visible loss of quality to the images.Smaller image sizes are useful for faster loading of your website which your visitors will value (especially those on mobile devices) and so will search engines.

Exhale requires an account key which you can get from the [exhale website](https://pressbro.com/exhale) free of charge. First 1000 compressions are on us.

Features:

- JPG/JPEG and PNG compression
- Up to 80% smaller images
- First 1000 compressions completely free

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/exhale-by-pressbro` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Media->Compress Images to start using the plugin.
